## To run the project

- ### Rename environment.prod-example.ts to environment.prod.ts on src/environments

- ### Change SEARCH_API and SEARCH_ENGINE to yours

- ### Install packages
  
  <b>npm install</b> or <b>yarn</b>
  
- ### Run server

  <b>ng serve</b> or <b>yarn start</b>
